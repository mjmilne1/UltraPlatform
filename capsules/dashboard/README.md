# Capsules Dashboard

Professional React dashboard for the Capsules Portfolio Platform.

## Features

- 💼 View all capsules and their goals
- 📊 Interactive portfolio allocation charts
- 💰 Transaction history and management
- 📈 Performance tracking
- ⚖️ One-click portfolio rebalancing
- ➕ Create new capsules

## Quick Start
```bash
# Install dependencies
npm install

# Start development server
npm start
```

Dashboard will be available at http://localhost:3000

## API Connection

The dashboard connects to the Capsules API at `http://localhost:8000`

Make sure your API is running before starting the dashboard.

## Tech Stack

- React 18
- Recharts (charts)
- Axios (API calls)
- Responsive design
